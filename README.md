Purple Layout Project

This project showcases a simple and modern layout created using HTML and CSS. The design is based on a grid-style structure with different shades of purple to highlight visual balance, spacing, and color contrast.

Features

Clean grid-based layout

Multiple color blocks with smooth contrast

Fully responsive design

Simple, beginner-friendly structure


Technologies Used

HTML5

CSS3
